# Autoencoders

Implementations of several autoencoder architectures using `tensorflow.keras`.

## Files

- `0-vanilla.py`: Vanilla (fully-connected) autoencoder
- `1-sparse.py`: Sparse (L1-regularized latent) autoencoder
- `2-convolutional.py`: Convolutional autoencoder
- `3-variational.py`: Variational autoencoder
