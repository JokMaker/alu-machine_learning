# Classification

This directory contains neural network implementations for binary and multiclass classification tasks.

## Files

- `0-neuron.py` to `7-neuron.py`: Single neuron implementations
- `8-neural_network.py` to `15-neural_network.py`: Neural network with one hidden layer
- `16-deep_neural_network.py` to `28-deep_neural_network.py`: Deep neural network implementations
- `24-one_hot_encode.py`, `25-one_hot_decode.py`: One-hot encoding utilities
