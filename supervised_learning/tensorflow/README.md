# TensorFlow

This directory contains TensorFlow implementations for neural network operations.

## Files

- `0-create_placeholders.py`: Creates placeholders for input data and labels
- `1-create_layer.py`: Creates a layer for a neural network
- `2-forward_prop.py`: Creates forward propagation graph
- `3-calculate_accuracy.py`: Calculates accuracy of predictions
- `4-calculate_loss.py`: Calculates softmax cross-entropy loss
- `5-create_train_op.py`: Creates training operation using gradient descent
- `6-train.py`: Builds, trains, and saves a neural network classifier
- `7-evaluate.py`: Evaluates the output of a neural network

## Requirements

- Python 3.5+
- TensorFlow 1.x
- NumPy
